Lesson 1: Introduction to HTML

What is HTML?

HTML stands for HyperText Markup Language. It's a programming language used to create and structure content on the web.

HTML Definition

HTML is a:

  • Markup Language: Uses tags to wrap around content.
  • HyperText: Links text, images, and other media.
  • Programming Language: Follows rules (syntax) to create web pages.

HTML Purpose

HTML's main goal is to provide a structure for web pages, making it possible for:

  • Web browsers to display content.
  • Search engines to understand website content.
  • Developers to create interactive web applications.

HTML Basics

To start building with HTML, you'll need to learn:

  • HTML Syntax: Rules governing HTML code structure.
  • HTML File Structure: Organization of HTML files.

HTML Syntax

HTML syntax refers to the rules governing the structure and organization of HTML code.

Basic Components:

  • Elements: Represented by < > symbols.
  • Attributes: Provide additional information about elements.
  • Content: Text or media within elements.

HTML File Structure

				
<!DOCTYPE html>

<html>

<head>

	<title>Page Title</title>

	<meta charset="UTF-8">

	<meta name="viewport" content="width=device-width, initial-scale=1.0">

</head>

<body>

	<!-- Content goes here -->

</body>

</html>
