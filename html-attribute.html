Lesson 3: HTML Attributes

What are HTML Attributes?

HTML attributes provide additional information about an HTML element.

They are added to the opening tag of an element.

Types of HTML Attributes

Required Attributes

Some attributes are required for certain elements.

Example: <img src="image.jpg" alt="Image description">

Optional Attributes

Most attributes are optional.

Example: <a href="#" title="Link title">Visit Our Website</a>

Boolean Attributes

Some attributes have only two values: true or false.

Example: <input type="checkbox" checked>

Common HTML Attributes

Link Attributes

href - specifies the link URL

title - specifies the link title

target - specifies the link target

Image Attributes

src - specifies the image URL

alt - specifies the image description

width and height - specify the image size

Table Attributes

border - specifies the table border

cellpadding - specifies the cell padding

cellspacing - specifies the cell spacing

Form Attributes

action - specifies the form action

method - specifies the form method

name - specifies the form name

Best Practices

1. Use lowercase attribute names.

2. Use quotes around attribute values.

3. Avoid duplicating attributes.

Example Code


<!DOCTYPE html>

<html>

<head>

<title>HTML Attributes</title>

</head>

<body>

<a href="#" title="Link title">Visit Our Website</a>

<img src="image.jpg" alt="Image description" width="200" height="150">

<table border="1" cellpadding="5" cellspacing="0">

<tr>

<th>Header</th>

</tr>

<tr>

<td>Cell</td>

</tr>

</table>

<form action="#" method="post" name="myForm">

<label>Name:</label>

<input type="text" name="name">

</form>

</body>

</html>

Exercises

1. Create a web page with links, images, and tables.

2. Add attributes to the elements.

3. Experiment with different attribute values.
