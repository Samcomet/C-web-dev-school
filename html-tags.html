Lesson 2: HTML Tags

What are HTML Tags?

HTML tags are like labels that tell web browsers how to display content on a web page. They're surrounded by < > symbols and usually come in pairs.

Types of HTML Tags

Paired Tags

Have an opening and closing tag

Surround content

Example: <p>This is a paragraph</p>

Self-Closing Tags

Have only one tag

Do not surround content

Example: <img src="image.jpg" />

Empty Tags

Have only one tag

Do not surround content

Example: <hr>

Basic HTML Tags

Headings

<h1> - Main heading

<h2> - Subheading

<h3> - Sub-subheading

Text

<p> - Paragraph

<span> - Small text section

Links

<a> - Hyperlink

Images

<img> - Image

Lists

<ul> - Unordered list

<ol> - Ordered list

<li> - List item

Divisions

<div> - Content section

Semantic HTML Tags

Structural Elements

<header> - Page header

<footer> - Page footer

<nav> - Navigation menu

<main> - Main content

<section> - Content section

<article> - Article

<aside> - Sidebar

Best Practices

1. Use lowercase tags.

2. Close tags in reverse order.

3. Use semantic tags for better structure.

Example Code


<!DOCTYPE html>

<html>

<head>

<title>HTML Tags</title>

</head>

<body>

<h1>Main Heading</h1>

<p>This is a paragraph.</p>

<a href="#">Visit Our Website</a>

<img src="image.jpg" alt="Our Company Logo">

<ul>

<li>Item 1</li>

<li>Item 2</li>

</ul>

<div>This is a division.</div>

<span>This is a span.</span>

</body>

</html>
